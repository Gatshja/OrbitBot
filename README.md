# OrbitBot

OrbitBot is a Discord bot dedicated to providing fascinating space-related data and information.

## Owners

- Armsix Production
- Robloxbot Team

## Features

- `/apod`: Fetches NASA's Astronomy Picture of the Day.
- `/iss-location`: Gets the real-time location of the International Space Station.
- `/about`: Provides details about the bot and its creators.

## Setup

1. Clone the repository.
2. Install dependencies: `npm install`
3. Create a `.env` file with your `DISCORD_TOKEN` and `CLIENT_ID`.
4. Run the bot: `node index.js`
